# API structure with Express.js

## Technologies Used

- Express 4.18.x
- Mongoose 6.6.x
- Redis 4.3.x

## How to use

- Backend

    ```bash
    > cd back-end
    > cp .env.example .env
    > npm start
    ```

- Frontend

    1. cd front-end
    2. use `Live Server` extension in VS Code to test api with CORS
